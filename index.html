Summary: Latitude vs. X

The goal of the project was to use our recent knowledge about Python requests, APIs, and JSON traversals to answer the question: "What's the weather like as we approach the equator?"

A Python script was created that analyzed a file of more than 500 cities around the world and generated a separate file that used the OpenWeatherMap API to generate weather information for those cities. A series of scatter plots were then created to showcase the following relationships: Max Temperature (F) vs. Latitude Humidity (%) vs. Latitude Cloudiness (%) vs. Latitude Wind Speed (mph) vs. Latitude The city data is included in this site, as well as the scatter plots created along with descriptions describing the trends seen in the data.
